Fullstack Project Experiene

Design and Implementation of a High-Concurrency Online Learning Platform --- BiliBili --- Software Engineer

Technology Used: Distributed System, Golang, Vue.js, SQL, Docker, Portainer, Redis, Alibaba Cloud, GPT API API, Axios, Nginx

Link: http://www.onlinecslearning.com

Date: 09/2021 - 12/2022

  • Leveraging Alibaba Cloud, our platform effectively handles up to 100K concurrent requests which broadens learning opportunities for Chinese by offering translated open-source courses from globally renowned institutions like CMU, Stanford, and MIT.
  • Developed an integrated user interface using Vue.js, Go and Gin, supporting comprehensive CRUD operations, SQL and Redis were used for robust data management and caching, Axios ensured seamless frontend-backend communication, Nginx functioned as a reverse proxy, and the ChatGPT API handled language translations.
  • Used Docker and Portainer for container management, Ali ACR for image storage, Ali OSS for video storage, and Github for code management.

Full Stack Web Development for Car Selling Company --- AeroAutoHub --- Software Development Engineer & Cofounder

Technology Used: Javascript, Github, Node.js, Express, MongoDB, GoogleStrategy, Passport

  • Used Node.js for the backend development, MongoDB for data storage, React.js for frontend, and Github for team collaborations.
  • Used Express for router swift and designed Admin and Client's Login system with Passsport and GoogleStrategy for security.
  • Implemented User-Based Collaborative Filtering recommendation system for car selling.
  • Contributed to Aero Auto Company's milestone of selling over 30 cars within a single month.
